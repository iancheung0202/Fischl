## Common Features

We provide you the countless useful features in our bot!

**Tickets** - We provide highly customizable private support channels, which we call tickets, between your staff team and individual who look for help.

**AI Assistance** - Want to obtain help or ask questions regarding Genshin, the server, or anything else? Just hit the bot up!

**Topic** - Try to keep your server chat engaged and active but got nothing to say? The bot issues random Genshin Impact topics to chat on constantly.

**Sticky Notces** - This function is especially useful when you need to tell members how to use a channel or what they need to know before going ahead and sending any messages.

**Genshin Texts** - Want some Genshin-styled text but don't want to dive into any graphic design software? Fischl automatically creates any texts with any color within seconds!
